# Анализ зарплат программистов с HeadHunter и SuperJob

Данная программа предназначена для сбора и анализа информации о зарплатах
программистов по различным языкам программирования с двух популярных сервисов
поиска работы:

* HeadHunter

* SuperJob

Программа автоматически:

- Получает данные о вакансиях через публичные API HeadHunter и SuperJob

- Фильтрует вакансии по языкам программирования

- Рассчитывает среднюю заработную плату

- Подсчитывает количество найденных и обработанных вакансий

- Отображает результаты в виде таблиц в терминале

***

## Как установить

Перед началом работы необходимо установить зависимости проекта.

В корневой директории выполните команду:

```
$ pip install -r requirements.txt
```

Документацию используемых библиотек можно найти на сайте [PyPI](https://pypi.org/).

***

## Настройка переменных окружения

Для работы с API SuperJob требуется API-ключ.

Создайте файл `.env` в корневой директории проекта и добавьте в него:

```env
SJ_KEY=ваш_api_ключ_superjob
```
* Для HeadHunter авторизация не требуется — используется публичный API.

***

## Как использовать

Запустите программу командой:

```
python main.py
```

После запуска в терминале будут выведены две таблицы:

1. Статистика зарплат программистов с HeadHunter
2. Статистика зарплат программистов с SuperJob

Данные выводятся отдельно для каждого языка программирования.

***

## Формат выводимых данных

Для каждого языка программирования отображается:

* Количество найденных вакансий
* Количество вакансий, по которым удалось рассчитать зарплату
* Средняя заработная плата (в рублях)

## Что делает программа:

```Python
get_vacancies_hh()      – получает список вакансий с HeadHunter по заданному языку программирования

predict_rub_salary_hh() – рассчитывает зарплату в рублях для одной вакансии c HeadHunter на основе диапазона salary_from / salary_to

get_hh_salaries()       – собирает статистику по зарплатам с HeadHunter:количество найденных вакансий, количество обработанных вакансий и среднюю зарплату по каждому языку программирования


get_vacancies_sj()      – получает список вакансий с SuperJob по заданному языку программирования с использованием API-ключа

predict_rub_salary_sj() – рассчитывает зарплату в рублях для одной вакансии SuperJob на основе полей payment_from / payment_to

get_sj_salaries()       – собирает статистику по зарплатам с SuperJob:количество найденных вакансий, количество обработанных вакансий и среднюю зарплату по каждому языку программирования

make_table()            – формирует таблицу для вывода в терминал на основе статистики по языкам программирования
```
